# 🧠 JobScraper – Internshala & Indeed Job Listings

A simple yet effective Python web scraper that collects job listings from platforms like **Internshala** and **Indeed**, using `BeautifulSoup` and `requests`. Filter jobs by keyword, location, or category and get a clean output of relevant opportunities!

---

## 🚀 Features

- 🔍 Scrape jobs based on custom search terms
- 🌍 Location-based filtering
- 📋 Output in readable format (JSON, CSV, or console)
- ⚡ Built using Python, BeautifulSoup, and Requests
- 🔄 Easily extendable for other platforms

---

## 🛠️ Tech Stack

- **Python 3.x**
- **BeautifulSoup**
- **Requests**
- **pandas** (for optional CSV export)
